# Miraç Su Tesisatı & Arıtma Sistemleri

Full Laravel Web Application with Front-End Pages and Admin Panel.

## Requirements
- PHP >= 8.1
- PDO SQLite Extension

## Quick Start
```bash
php artisan serve
```
Open `http://127.0.0.1:8000` in browser.

## Admin Panel
- **URL:** `http://127.0.0.1:8000/admin`
- **Username:** `admin`
- **Password:** `password123`
